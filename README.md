# DIMMER---TRIAC
Using PIC16F886 , using zero crossing
IDE : PICC
# Have 4 button
1: ON/OFF
2: AUTO/HAND
  2.1: AUTO
        Auto running up and down 0% - 100%
  2.2: HAND
        2.2.1: UP
        2.2.2: DOWN
